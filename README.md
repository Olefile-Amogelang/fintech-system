 Fintech Transaction Monitoring System

 Overview
This project simulates a fintech transaction monitoring system that ingests , processes, and analyzes financial transactions to detect  anomalies and potential fraud in near real time world.

 Demonstrates an end to end data pipeline includin;
- Data ingestion and cleaning
- Fraud detection logic
- Database storage
- Dashboard for monitoring
- ETL processing

Business Value
-Reduces financial risk through early fraud detection
-Automates transaction monitoring workflows
-Enables data driven decision making for compliance teams
 How to Run
pip install -r requirements.txt
python run_pipeline.py
streamlit run dashboard/app.py
